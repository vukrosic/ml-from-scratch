# Adam Optimizer From Scratch

Momentum + RMSprop + bias correction. Build it, then see why it works.

## What we build
- SGD baseline
- Momentum
- RMSprop
- Adam (combining both + bias correction)
- Loss landscape visualization showing optimizer paths

- `visualize.py` — optimizer trajectories on 2D loss surface
