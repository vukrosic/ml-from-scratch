# Softmax From Scratch

Why softmax works, the numerical stability trick, and temperature scaling.

## What we build
- Naive softmax (overflows)
- Stable softmax (subtract max)
- Temperature scaling
- Jacobian of softmax

## Files (planned)
- `softmax.py` — all variants
- `jacobian.py` — compute and visualize the Jacobian
- `benchmark.py` — compare against torch.softmax
