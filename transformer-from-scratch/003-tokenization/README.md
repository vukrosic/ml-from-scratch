# Tokenization From Scratch

BPE tokenizer built from nothing. Understand why "token" ≠ "word".

## What we build
- Character-level baseline
- Byte Pair Encoding (BPE) training loop
- Encode/decode with learned merges
- Compare against tiktoken

## Files (planned)
- `bpe.py` — train and use BPE
- `compare.py` — vs tiktoken output
